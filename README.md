# bokbokbok

## Overview

**bokbokbok** is a python package that enables you to use Custom Loss Functions and Evaluation Metrics for XGBoost and LightGBM.
Main features:

- Weighted Cross Entropy
- Focal Loss
- Squared Log Error
- Log Cosh Loss
- F1 score

## Installation

```bash
pip install bokbokbok
```
